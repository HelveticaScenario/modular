Project: Modular Synthesizer (Electron app + Rust DSP). Real-time modular synth with JS DSL creating PatchGraph JSON sent via Electron IPC to Rust audio engine. Renderer shows Monaco editor + oscilloscope overlays. Tech stack: Rust (modular_core, modular, modular_derive, mi-plaits-dsp-rs), Electron + React + TypeScript renderer, N-API bindings. Key flow: DSL -> PatchGraph -> IPC -> Rust validation -> audio thread (cpal).
